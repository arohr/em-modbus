# Modbus

Modbus Client and Server for eventmachine.
It's a pure Ruby implementation.

## Installation

Add this line to your application's Gemfile:

    gem 'em-modbus'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install em-modbus
